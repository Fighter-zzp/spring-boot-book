# 个人博客练习
## 用户模块
### Thymeleaf
**四个公共模块**:
- 公共头模块：主要用于引用CSS资源和头部信息
- 公共导航模块：主要引入导航栏
- 公共底部模块：主要用于底部信息
- 公共js模块：用来引入js

### 拦截器
- 使用渲染时增强，为网页加入底部信息
- 访问次数加载

### 定时器
一段时间内，加载访问记录

### 初始化
无访问记录和webConfig信息时，加载

### 博客页
分页展示博客中内容

### 搜索页
根据关键词和内容搜索已发布内容

### 详情页
展示文章详情

### 联系页
保存用户消息消息（只是单纯的保存，无发邮件）
